# WPF Employee App

This is a **WPF (Windows Presentation Foundation)** application built using **C#**. It calculates an employee's **years of experience** and the **number of days** they’ve worked based on their **start date** and the **current date**.

## Overview

The **WPF Employee App** is a simple application developed to assist in calculating an employee's **working experience** at a company. The app allows the user to enter the employee's **start date** and **salary**, and it computes:
- The number of **years** the employee has worked.
- The total **number of days** the employee has been with the company.
- Based on this, the app can also calculate additional data like **average salary** per year.

## Features

- **User-Friendly Interface**: Built using **XAML** and **C#** to create an intuitive, desktop-based interface.
- **Calculate Work Experience**: Automatically calculates how many years and days the employee has worked based on their start date.
- **Salary Calculation**: Compute the salary per year based on the inputted salary.
- **Data Validation**: Ensures that the user inputs valid data (e.g., valid dates, salary amounts).

## Prerequisites

Before running the application, ensure you have the following installed:

- **Visual Studio** (with .NET SDK installed)
- **.NET 6.0** or higher (for building and running the application)

### Optional (for building the app):
- **Git** (to clone the repository)

## Installation

Follow these steps to get your copy of the WPF Employee App up and running:

1. **Clone the Repository**:
   If you have **Git** installed, you can clone this repository by running the following command in your terminal:

   ```bash
   git clone https://github.com/YOUBY-25/WPFAPP1.git

Alternatively, you can download the repository as a ZIP file by clicking the green Code button on the GitHub page and selecting Download ZIP.

Open the Project in Visual Studio:
After cloning or downloading the repository, open Visual Studio.
Click File > Open > Project/Solution and select the WPFAPP1.sln file.
Build the Project:
Once the project is loaded, build the solution by pressing Ctrl + Shift + B or selecting Build > Build Solution from the top menu.
Usage
Running the Application:

Once the project is built successfully, follow these steps to run the application:

Start the Application: Press F5 or click on the green play button in Visual Studio to start the application.
Enter the Employee Details:
Input the employee's start date and salary.
View the Calculations:
The app will display the employee's total years worked, days worked, and other useful details based on the data entered.
