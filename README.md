# Employee App

This is a WPF application built using C#.

## Description
This app calculates the years of experience and days worked for an employee.

## How to Use
1. Clone this repository: `git clone https://github.com/YOUBY-25/WPFAPP1.git`
2. Open the project in Visual Studio or another C# IDE.
3. Run the application.
