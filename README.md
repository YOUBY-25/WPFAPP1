# WPF Employee App

This is a simple WPF (Windows Presentation Foundation) application built using **C#**. It calculates an employee's years of experience and the number of days they've worked based on their start date.

## Prerequisites
- **Visual Studio** (with the .NET SDK installed)

## How to Use
1. Clone or download this repository.
2. Open the `WPFAPP1.csproj` file in **Visual Studio**.
3. Press **F5** or click **Run** to build and run the application.

## Features
- Calculate employee experience based on their start year.
- Calculate the number of days worked since they started.
